U盘+Ventoy+Ubuntu

准备：

  • 一台 Windows / Linux 电脑
  • 一个 Ubuntu 或其他 Linux 发行版镜像，尽量选择 LTS （长期支持版）
  • 一个 U 盘，容量至少 64 G，传输速度越快越好（也可选择移动硬盘）
  • Ventoy 软件，下载链接
  • 虚拟机，如 VM VirtualBox
